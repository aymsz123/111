# AI课程期末项目报告：RAG课程文档智能问答助手
## 1. 选题与需求分析
本项目为人工智能课程学习开发课程文档RAG智能问答助手，面向选课学生解决课程资料查阅繁琐、知识点复习效率低的问题，基于向量检索+大模型RAG技术实现课程知识库精准问答。AI承担文档向量检索、课程内容理解、开放式智能回答功能，相比固定脚本可适配各类课程开放性问题。

## 2. 系统架构与技术选型
### 系统架构
- 四层架构：前端交互层、文档处理层、向量知识库层、大模型RAG问答层
- 数据流向：课程文档→文本分块→Embedding向量入库→检索匹配→大模型Prompt增强回答
### 技术选型
- Python + Streamlit + LangChain + BGE Embedding + Chroma向量库 + LLM API
- 纯本地网页应用，无需额外硬件

## 3. 关键实现说明
1. 课程PDF文档解析、递归文本分块，保证上下文连贯性
2. 使用BGE中文Embedding做向量化，存入Chroma本地向量库做相似度检索
3. LangChain构建RAG检索问答链，通过Prompt约束模型仅基于课程原文回答，附带原文引用
4. Streamlit实现可视化交互页面，支持对话历史、文档上传功能

## 4. 测试结果与局限性
### 测试结果
课程基础知识点问答准确率较高，可正确引用课程原文生成复习资料；基础问答延迟2秒左右。
### 局限性
1. 纯图片课件内容无法识别（未加入OCR视觉模型）
2. 长公式、复杂数学推导回答精度不足
3. 依赖大模型API联网调用，离线使用受限
### 改进方向
后续加入OCR文档解析、微调小模型实现本地离线部署、增加课程习题自动生成功能

## 5. 个人收获与改进方向
### 收获
完整掌握RAG检索增强生成技术原理、LangChain框架使用、向量数据库应用、端到端AI网页应用开发流程；理解大模型幻觉问题的解决思路，完成从需求设计、编码实现、测试演示的完整工程闭环。
### 改进方向
1. 引入多模态模型支持图文课件
2. 做课程知识点记忆/自测功能
