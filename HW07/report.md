# HW07：胸部X光肺炎检测实验报告

## 1. 实验背景
胸部X光肺炎检测是医学影像分析的经典入门任务，本实验基于Kaggle公开数据集，使用卷积神经网络实现肺炎二分类，体会深度学习在医疗影像中的应用流程。

## 2. 数据集说明
- 数据集：Chest X-Ray Images (Pneumonia)
- 样本规模：训练集约5200张，测试集约600张
- 类别分布：训练集中肺炎样本占比约70%，存在类别不平衡问题
- 数据划分：按8:2从原训练集划分训练/验证集，原验证集样本并入测试集

## 3. 模型结构
本次实验使用简单CNN模型，结构如下：

| 层类型 | 输出尺寸 | 说明 |
|--------|----------|------|
| 输入层 | 1×150×150 | 灰度X光影像 |
| 卷积层1 | 32×150×150 | 3×3卷积+ReLU |
| 池化层1 | 32×75×75 | 2×2最大池化 |
| 卷积层2 | 64×75×75 | 3×3卷积+ReLU |
| 池化层2 | 64×37×37 | 2×2最大池化 |
| 卷积层3 | 128×37×37 | 3×3卷积+ReLU |
| 池化层3 | 128×18×18 | 2×2最大池化 |
| 全连接层1 | 256 | Dropout(0.5) |
| 输出层 | 1 | 二分类输出 |

## 4. 实验配置
- 优化器：Adam，学习率0.001
- 损失函数：BCEWithLogitsLoss
- 批次大小：32
- 训练轮数：10
- 数据增强：随机水平翻转、随机旋转

## 5. 实验结果
### 测试集指标
- 准确率（Accuracy）：约93%
- 精确率（Precision）：约92%
- 召回率（Recall）：约97%
- F1分数：约94%

## 6. 结果分析与思考
1. **指标分析**：模型在测试集上召回率较高，说明对肺炎样本的漏诊率较低。从医学诊断角度，召回率更为重要，因为假阴性（肺炎误判为正常）可能延误治疗，造成严重后果；而假阳性（正常误判为肺炎）仅需进一步检查，风险较低。

2. **数据增强的作用**：数据增强有效缓解了过拟合，提升了模型在测试集上的泛化能力。
